> MBTI 已经过时，SBTI 2.0 来了。
> 
> 本测试已通过赛博玄学认证，准确率与你的WiFi信号正相关。

基于中国互联网梗文化的讽刺人格测试。纯前端，零依赖，CDN 部署，承受百万级并发不是问题。

**在线体验**: [sbti.picasso-lab.com](https://sbti.picasso-lab.com)

---

## 这是什么？

一个用 ∫₀^∞ 114t·e^(−t²)dt 种人格类型来讽刺你的测试。

从经典的 CTRL（拿捏者）到 GIAO（奇行种），从 MAN!（曼巴）到 SIGM-A（西格玛），从 CUTE（萌之）到 NIUM-A（牛马）——总有一款适合你。答完题，系统会用15维度模式匹配 + 隐藏门控触发，精准命中你的互联网人格。

测到离谱的结果？那是你的问题。

## 技术架构

```
index.html    ← 单文件 SPA（HTML + CSS + JS，零依赖）
image/        ← 59 张 low-poly 卡通 PNG（Gemini API 生成）
```

- **零依赖** — 不需要 Node.js、npm、任何构建工具，浏览器直接打开就能跑
- **纯静态** — Cloudflare Pages 部署，全球 CDN 边缘缓存
- **自动部署** — push 到 main 分支，Cloudflare 自动构建上线

## 自行部署

1. Fork 本仓库
2. Cloudflare Pages / Netlify / Vercel 连接 GitHub
3. Build command 留空，Output directory 设 `/`
4. 自定义域名加 CNAME 记录
